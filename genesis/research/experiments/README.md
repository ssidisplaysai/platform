# Experiments Framework

Status: Approved
Classification: Genesis Research Process

## Purpose

This directory defines how research experiments are proposed, executed, and recorded.
Experiments are the primary mechanism for testing hypotheses before specification candidacy.

## Experiment Creation Rules

- every experiment must map to a research area
- every experiment must state a falsifiable hypothesis
- every experiment must declare required evidence inputs
- every experiment must include repeatability notes
- every experiment must record both success and failure outcomes

## Minimum Completion Criteria

An experiment record is complete only when:
- method and inputs are documented sufficiently for reproduction
- evidence references include source and lineage
- results include confidence and limitations
- next step is defined even when the hypothesis fails
