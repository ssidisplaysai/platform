# Genesis Constitution

Status: Active constitutional baseline reference  
Owner: Architecture Review Board

## Constitutional Principles

1. Architecture precedes implementation.
2. Business metadata is the source of truth.
3. Determinism, governance, and auditability are mandatory.
4. Architectural changes require formal RAR -> ARD -> ADR process.
5. Canonical model boundaries must not be violated.
