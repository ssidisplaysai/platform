# GRT-0001 Architecture Delta

Scope:
- Added a dedicated runtime-kernel module under src/runtime/kernel for execution semantics of Enterprise Runtime IR.

Added components:
- Kernel orchestration: RuntimeKernel
- Context model: RuntimeContext and snapshot types
- Lifecycle control: RuntimeStateMachine
- DI and graph checks: DependencyContainer, RuntimeKernelValidator
- Registries: ServiceRegistry, ModuleRegistry, PluginRegistry
- Eventing and scheduling: EventDispatcher, RuntimeScheduler
- Operability: HealthManager, RecoveryManager, RuntimeConfigurationManager

Boundary decisions:
- Runtime kernel consumes Enterprise Runtime IR generated by GCC-1008.
- Compiler pipeline behavior remains unchanged.
- Runtime kernel is additive and isolated from existing runtime core in src/core/runtime.
- Runtime kernel governance closure approved by GAR-0020 and GD-0011.
