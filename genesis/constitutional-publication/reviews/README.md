# Reviews Domain

## Purpose

This domain contains constitutional review infrastructure, reusable review classes, and recovery-review preparation artifacts used to assess constitutional corpus admission and publication legitimacy.

## Scope

This domain governs:
- reusable review templates
- review gate preparation artifacts
- corpus admission review preparation

This domain does not create new constitutional meaning.
