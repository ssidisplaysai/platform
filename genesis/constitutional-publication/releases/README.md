# Releases Domain

## Purpose

This domain contains canonical constitutional release definitions derived from the certified constitutional corpus.

## Scope

This domain governs:
- constitutional release definitions
- included artifact sets
- included certification references
- release baseline declarations

## Relationship to Certification

Constitutional releases are issued only on the basis of certification records and readiness evidence.
