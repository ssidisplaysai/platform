# Genesis OS Principles

## Architectural Principles

- Follow Clean Architecture.
- Keep business logic in services.
- Keep data access in repositories.
- Keep runtime behavior in the runtime layer.
- Keep UI presentation-focused.

## Engineering Principles

- Prefer clear abstractions over shortcuts.
- Use strict typing and explicit contracts.
- Write complete, reviewable implementations.
- Improve the platform for the next decade, not just the next release.
