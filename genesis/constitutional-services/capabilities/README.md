# Constitutional Service Capabilities

Identifier: GCSA-0002
Title: Genesis Constitutional Service Capability Model
Status: Draft
Authority: Foundation Authority
Review Target: GAR-0043

## Purpose

This domain defines the complete capability model for Genesis Constitutional Services.

Capabilities represent business behaviors owned by constitutional services and remain independent of implementation, transport, runtime, persistence, and technology.

## Authoritative Dependencies

The following artifacts are immutable baseline dependencies for this capability model:
- GCR-1.0
- AFR-0007
- GCCR-0001
- GCSA-0001

## Scope

This domain defines:
- capability catalog by service
- capability ownership rules
- capability dependency and composition model
- capability lifecycle and authority hierarchy
- capability evolution, versioning, and stability principles

## Constraint

This domain is architecture only and introduces no operations, APIs, runtime behavior, persistence design, messaging design, or implementation technology.
