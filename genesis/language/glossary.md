# Genesis Business Glossary

## Terms

- Business Object: a core domain entity in Genesis OS.
- Relationship: a defined connection between two business objects.
- Event: a meaningful occurrence that changes platform state or triggers behavior.
- Lifecycle: the state progression of a business object over time.
- Status: a named state within a lifecycle.
- Capability: a reusable business function exposed by the platform.
- Definition: the metadata description of a business object or behavior.

## Usage Rule

Glossary terms must be used consistently across documentation, runtime behavior, and module design.
