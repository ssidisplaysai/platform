# Genesis Repository Overview

## What This Repository Contains

1. Architecture records and governance artifacts under docs/architecture.
2. Compiler platform and tooling under tools/genesis.
3. Source modules and platform code under src.
4. Business definitions under definitions/entity.
5. Genesis standards and language documents under genesis.

## Canonical Paths

1. Architecture baseline: docs/architecture
2. Governance baseline: docs/governance
3. Engineering reports: docs/reports
4. Onboarding guides: docs/onboarding
5. Readiness checklists: docs/readiness

## Source Of Truth Rules

1. Primary working tree is repository root.
2. Mirror paths (for example platform-ssi-discovery) are non-authoritative unless explicitly designated for migration.
3. Architecture decisions are authoritative only when ratified in board-approved ARD/ADR records.
