M1.8 strictly adheres to architectural constraints:

- ✅ **Graph is NOT a database**: No persistence, no indexing, no query engine
- ✅ **Graph is NOT a runtime model**: No lazy evaluation, no dynamic modification during execution
- ✅ **Graph is NOT an application model**: No UI concerns, no business logic execution

All nodes and edges carry canonical identities assigned in M1.7:

- **Version-Aware**: All nodes/edges track GPS standards versions
- **Repeatable**: Repeated compilation of identical identities produces identical graphs

### No Synthesis Constraint

Graph construction operates on pure projection principles:

- ✅ Every node originates from a `BusinessGenomeIdentity` of kind `semantic-object`
- ✅ Every edge originates from a `BusinessGenomeIdentity` of kind `semantic-relationship`
- ❌ NO nodes are created beyond identity count
- ❌ NO edges are created beyond identity count  
- ❌ NO synthesized relationships inferred
- ❌ NO missing objects created

## 3. Graph Model

### BusinessGenomeNode

Each node is an immutable projection of a semantic object with complete provenance:

```typescript
interface BusinessGenomeNode {
  id: string;  // Canonical identity (bg.object_<class>_<SHA256>_v1)
  semanticClass: string;  // From identity (customer, supplier, product, etc.)
  canonicalDesignation: string;  // From consolidated semantic
  sourceIdentityId: string;  // Traceability to M1.7 identity
  sourceConsolidatedSemanticId: string;  // Traceability to M1.5 semantic
  provenanceReferences: readonly string[];  // Full provenance trail (sorted)
  evidenceLineage: readonly string[];  // All evidence items (sorted)
  sourceEvidenceIrIdentity: string;  // Root evidence IR identity
  constructedAt: string;  // Deterministic timestamp (2024-01-01T00:00:00Z)
  certainty: { state: "certain" | "uncertain"; confidence: number };
  validationStatus: { valid: boolean; violations: readonly string[] };
  graphConstructionContext: GraphConstructionContext;  // Rule/version info
  diagnostics: readonly CompilerDiagnostic[];
}
```

**Invariants**:
- Every node is immutable (readonly fields throughout)
- All collections within nodes are sorted for reproducibility
- Nodes preserve all provenance from earlier passes
- Nodes contain no derived or computed fields

- Edges maintain full traceability to source relationships
